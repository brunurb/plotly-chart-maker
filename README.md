# ChartMaker - CSV to Chart Converter

Online tool to create beautiful charts from CSV files.

## Features
- Multiple chart types (Bar, Line, Scatter, Pie, Area)
- Color palette selection
- Export as PNG, JPEG, SVG, PDF
- Batch processing
